# yl-cli

![Npm 版本](https://img.shields.io/badge/yl-cli_v0.0.1-green)

## 🎄 简介

- 用于快速搭建前端项目的命令行工具

## 🔥 模版

- ✅ 快速生成 [Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock 的模版()](https://gitee.com/sohucw/admin-pro)

## ✨ 安装

```bash
npm install yl-cli -g
```

## ❤️ 使用

```bash
#使用模版创建新项目
yl create

# 执行项目文件名称并选择模版创建新项目
yl create [projectName]

# 查看脚手架版本
yl -v
yl --version
```
